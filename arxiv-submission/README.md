# ClawChain arXiv Submission

LaTeX source for the ClawChain whitepaper, formatted for arXiv submission.

**Title:** ClawChain: A Layer 1 Blockchain for Autonomous Agent Economies  
**Categories:** cs.CR (Cryptography and Security), cs.DC (Distributed, Parallel, and Cluster Computing)

## File Structure

```
arxiv-submission/
├── main.tex           # Main document (entry point)
├── abstract.tex       # Abstract
├── introduction.tex   # Section 1: Introduction
├── architecture.tex   # Section 2: Technical Architecture
├── tokenomics.tex     # Section 3: Economic Model
├── governance.tex     # Section 4: Governance
├── usecases.tex       # Section 5: Use Cases
├── security.tex       # Section 6: Security Analysis
├── roadmap.tex        # Section 7: Development Roadmap
├── conclusion.tex     # Section 8: Conclusion
├── references.bib     # BibTeX bibliography
├── figures/           # Figures directory
├── Makefile           # Build automation
├── arxiv-meta.txt     # arXiv metadata
└── README.md          # This file
```

## Compilation

### Prerequisites

- TeX Live 2023+ or equivalent LaTeX distribution
- Required packages: `amsmath`, `hyperref`, `natbib`, `booktabs`, `listings`, `cleveref`, `algorithm`, `algpseudocode`

### Build

```bash
# Full build (recommended)
make pdf

# Quick single-pass build
make quick

# Clean auxiliary files
make clean

# Package for arXiv upload
make arxiv
```

### Manual build

```bash
pdflatex main
bibtex main
pdflatex main
pdflatex main
```

## arXiv Submission

1. Run `make arxiv` to generate `clawchain-arxiv.tar.gz`
2. Upload the tarball to https://arxiv.org/submit
3. Select categories: **cs.CR** (primary), **cs.DC** (secondary)
4. Fill in metadata from `arxiv-meta.txt`

### Important Notes

- arXiv requires the `.bbl` file (generated by BibTeX), not just `.bib`
- The `make arxiv` target includes the `.bbl` file automatically
- Verify the PDF renders correctly on arXiv's compilation system
