# CLAWCHAIN-DID-INTEGRATION

## Introduction

This document describes the integration of CLAWCHAIN with DID (Decentralized Identifier) registration.

## Configuration

The configuration for the discoverer is defined in the `DiscoveryConfig` struct.

## Testing

The discoverer can be tested using the `TestCheckReachable_Healthy` and `TestCheckReachable_NodeDown` functions.

## Implementation

The implementation of the discoverer is in the `internal/clawchain/discovery.go` file.
