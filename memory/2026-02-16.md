## GPU Server Swap Addition
**Time:** 19:29 - 19:34 AEDT
**Task:** Add 256GB swap to GPU server for LTX-2 model loading

### Issue
- LTX-2 checkpoint: 27GB
- System RAM: 16GB
- ComfyUI mmaps entire model before GPU load → RAM insufficient

### Solution
Created 256GB swapfile on /data2 (931GB USB HDD, 710GB free)

**Commands executed:**
```bash
sudo fallocate -l 256G /data2/swapfile
sudo chmod 600 /data2/swapfile
sudo mkswap /data2/swapfile
sudo swapon /data2/swapfile
echo '/data2/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```

**Result:**
- Before: 16GB RAM + 4GB swap = 20GB total
- After: 16GB RAM + 260GB swap = 276GB total ✅
- Permanent (survives reboot via /etc/fstab)
