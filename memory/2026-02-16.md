### Key Decisions

- **Unsigned by default:** Ship without code signing initially (users see warnings, can still install)
- **Auto-update off by default:** Notify about updates, don't auto-install
- **Background checks:** Every 24 hours (configurable)
- **Installation paths:**
  - Binary: `/usr/local/bin/evoclaw`
  - Config: `~/.evoclaw/` (user) or `/etc/evoclaw/` (system)
  - Service: systemd/launchd integration

### Lessons

- **Quick iteration wins:** Built full mainstream release infrastructure in 25 minutes
- **CI automation is key:** One tag push = all platforms built
- **Code signing is expensive:** Apple ($99/year) + Windows ($200-400/year) = ~$500/year
- **Installers matter:** MSI/DMG >> raw binaries for mainstream users
- **Documentation sells:** DESKTOP-RELEASE.md makes it real
