## Agent Patterns Integration (sub-agent: implement-agent-patterns, in progress)
Bowen approved incorporating best patterns from agent framework analysis into EvoClaw:
1. **WAL Protocol** — write-ahead log, agents persist state before responding (prevents losing corrections)
2. **Working Buffer** — explicit compaction danger-zone handling
3. **VBR (Verify Before Reporting)** — verify mutations actually improve before recording
4. **ADL (Anti-Divergence Limit)** — cap mutation distance from original genome, force simplification
5. **VFM (Value-For-Money) scoring** — reject mutations where fitness_improvement/complexity_cost < threshold
